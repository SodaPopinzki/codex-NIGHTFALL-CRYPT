# Nightfall Crypt - Gothic Vampire Survivors
## Tech: Phaser 3.88 (npm), Vite, JavaScript
## Game Type: Top-down auto-battler with wave survival
## Style: Dark gothic pixel art aesthetic (Castlevania inspired)
## Rules:
- Mobile-first with virtual joystick for touch + WASD/Arrow for desktop
- 60fps target on mid-range phones
- Keep files under 400 lines, split into modules
- Arcade Physics for all collisions (no Matter.js)
- All game balance numbers in src/config/GameConfig.js
- Use object pooling for projectiles and enemies
- Test npm run build before every commit
